For Arduino Nano R4

Emulates mouse to click on a specific absolute screen location when physical button pressed.

Requires library https://github.com/jonathanedgecombe/absmouse

To program button positions, hold down REC button for 10 seconds. Mouse will move to top left of screen
and start to jiggle. Use sliders as X/Y to position the mouse pointer where required for each button then 
press the physical hardware button to record the position.

Move to REC button screen location and hold down REC button again for 10 seconds to exit program mode.
